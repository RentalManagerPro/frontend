# Setup project

## Install node and npm
```bash
brew install node
```

## Check if node and npm are installed
```bash
node -v
npm -v
```

## Enable corepack
```bash
corepack enable && corepack enable npm
```

## Install watchman
https://facebook.github.io/watchman/docs/install#buildinstall

## Install dependencies
```bash
npm install
```

## Run the project
```bash
npm run web
```
